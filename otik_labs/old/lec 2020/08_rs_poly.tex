\section{Корректирующий код}
\begin{frame}{\insertsection}

Минимальная единица передачи по каналу связи "--- \termin{символ} (элемент некоторого поля).

Каждый символ может быть искажён при передаче независимо от других.

При корректирующем кодировании в каждое \termin{кодовое слово}, помимо \termin{информационных} символов, вводят \termin{проверочные}, или корректирующие.

\end{frame}

\subsection{Циклические коды}
\begin{frame}{\insertsubsection}
\setlength{\parskip}{0.1\parskip}

Циклическая перестановка символов в кодовом слове дает другое допустимое слово того же кода.
\vfill
% Пусть кодовое слово $C_2$ получено из $C_1$ путем циклического сдвига символов влево

$C_1 = (c_0, c_1, \ldots c_{n-1})$

$C_2 = (c_{n-1}, c_0, c_1, \ldots c_{n-2})$

Таким образом, $C_2 = x\cdot C_1 - c_{n-1}\cdot (x^n - 1)$.
\vfill

Необходимо, чтобы $x^n - 1$ делился на $g(x)$.

Проверочный многочлен $h$:
% ????????????????????????????????/
% $h(x) = \dfrac{x^n - 1}{g ( x^{-1} ) \cdot x^{\deg(g)}}$
\rlap{$g(x)h(x)=0\mod x^n-1$}

 
$C(x)h(x)=a(x)g(x)h(x)=0\mod x^n-1$
\end{frame}

\section{Двоичные циклические коды}
\begin{frame}{\insertsection}

Символы двоичных кодов "--- элементы $\GF(2)$.

Циклическая перестановка: $C_2 = x\cdot C_1 - c_{n-1}\cdot (x^n - 1)$

Для символов из $\GF(2)$\\ либо $C_2 = x\cdot C_1$, 
\\либо $C_2 = x\cdot C_1- (x^n - 1)$,
что эквивалентно $C_2 = x\cdot C_1 + (x^n + 1)$;

$x^n + 1=g(x)\cdot h(x)$,
$h(x)$ "--- проверочный многочлен.
\end{frame}

Пусть $\beta$ "--- элемент поля \GF(q) порядка $n$ (обычно выбирается примитивный элемент). 
% Если $\beta$ "--- примитивный элемент, то его порядок равен $q - 1$.
% , то есть $\beta^{q-1}=1,\quad \beta^i \neq 1, 0 < i<q-1$. 
% Пусть $\beta$ "--- примитивный элемент.

Тогда 
% нормированный полином $g(x)$ минимальной степени над полем \GF(q), корнями которого являются $d - 1$ подряд идущих степеней $\beta^{l_0}, \beta^{l_0+1},...,\beta^{l_0+d-1}$ элемента $\beta$, является порождающим полиномом кода Рида"--~Соломона над полем \GF(q):
порождающий полином кода Рида"--~Соломона:

$g(x) = (x - \beta^{l_0})(x - \beta^{l_0+1})\dots(x - \beta^{l_0+d-1})$

где $l_0$ "--- некоторое целое число.
% (в том числе 0 и 1), с помощью которого иногда удается упростить кодер. 
Обычно $l_0 = 1$.  
$\deg(g(x)) = d - 1$.

\subsection{Длина кода}
\begin{frame}{\insertsubsection}
\begin{adjustwidth}{-0.5em}{-1.8em}
\setlength{\parskip}{0.2\parskip}

Длина полученного кода $n$,

минимальное расстояние $m$,

проверочных символов $\mu = m - 1 = \deg(g(x))$,  информационных символов $k = n - \mu = n - m + 1$. 
% Таким образом $m = n - k - 1$ и код Рида — Соломона является разделимым кодом с максимальным расстоянием (является оптимальным в смысле границы Синглтона).
\bigskip

Если $\beta$ "--- примитивный элемент \GF(q), то $n = q - 1$.
% % (количество информационных символов $k$ максимально).
Количество проверочных $\mu$ однозначно определяет $g(x)$.% и кодовое расстояние.

Исправляется до $\mu/2$ ошибок.

\end{adjustwidth}
\end{frame}

\subsection{Декодирование систематического кода Рида-Соломона}
\begin{frame}{\insertsubsection}
\begin{adjustwidth}{-0.5em}{-1.8em}
\setlength{\parskip}{0.5\parskip}
\small

Локатор ошибки "--- $X_i = \beta^\ell$ для $x^\ell$.
Многочлен локаторов $L(x) = (1 - xX_1)(1 - xX_2)...(1 - xX_u)$

\begin{itemize}
\item Остаток $e(x) = C(x) \mod g(x)$.

\item Синдром $S(x):$ $s_i = e(\beta^{i+1}) = C(\beta^{i+1})$.

\item Многочлен ошибок $W(x)$ "--- степень не превышает $u-1$, где~$u$ "--- количество ошибок ($u \leqslant \mu/2$),
причём $L(x)\cdot S(x) = W(x) \mod x^\mu$.

Метод Берлекампа-Месси.

\item Значения ошибок $Y_i = \frac{W(X_i^{-1})}{L'(X_i^{-1})}$.

\end{itemize}

\end{adjustwidth}
\end{frame}
