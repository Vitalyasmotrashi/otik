\nsubsection{Допуск для индивидуальщиков и~должников}

Для допуска к~курсу ОТИК (специфические коды для сжатия и~защиты от помех)
необходимо \textbf{подтвердить знакомство с~кодами, используемыми в~ЭВМ по умолчанию.}

Студенты, обучающиеся по плану группы, 
%и~завершившие курс ОргЭВМ\,[и\,Асм] "--- 
допущены к~изучению ОТИК, явное подтверждение не нужно.

Студентам, обучающимся по индивидуальному плану, а~также должникам,
перед защитой лабораторной работы \ref{lab_freq} (но, возможно, параллельно с~\ref{lab_intro}) необходимо:
\begin{itemize}
\item если в~ОРИОКС есть положительная оценка по курсу ОргЭВМ\,[и\,Асм] "--- предъявить её преподавателю;
\item если её нет "--- выполнить и~защитить вводную лабораторную \ref{lab_oevm} (у~преподавателя лабораторных работ) либо закрыть долг иным способом (у~лектора, см.~\repoUrlAsm).
% (оценивается как работа на одно занятие, графа «бонус (л/р)»).
\end{itemize}

Должникам, выполнившим \ref{lab_oevm} в~срок, долг по ОргЭВМ\,[и\,Асм] закрывается автоматически (если у~преподавателя лабораторных работ нет доступа "--- сообщите лектору, что долг снят).
