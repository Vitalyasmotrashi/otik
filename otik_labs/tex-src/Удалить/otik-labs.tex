\title{Основы теории информации и~кодирования}

\maketitle


Дерево оптимального префиксного кода должно одновременно обладать тремя свойствами:

\begin{anumerate}
\item длина ветви $a$ в~рёбрах должна стремиться к~$I_{\text{BP}}(a)$ в~битах;

\item сбалансированностью: для каждого родительского узла веса обеих дочерних ветвей должны стремиться к~равенству;

\item самые длинные ветви (длинные коды) соответствуют самым редким символам (следует из (a)).
\end{anumerate}
