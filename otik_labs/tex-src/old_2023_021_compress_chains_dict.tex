\section{Код Зива--Лемпеля--Велча, LZW}
% \subsection{Понятие}
\begin{frame}{\insertsection}
\begin{adjustwidth}{-5em}{-5em}

1984 г., Терри Велч (Terry Welch) по концепции LZ78:

\begin{enumerate}

\item Вначале словарь = первый уровень (все одиночные символы, $2^k$ штук для $k$-битного \mbox{символа=байта).}
Тогда корень можно не нумеровать (прикорневые нумеруем с~нуля).

% \vfill
% \item Номер шага кодирования/декодирования "--- 

\vfill
\item Кодирование: при добавлении узлу $P$ дочернего узла с~номером $m$ и~символом $c_i$%(новое слово $c_{i-r}c_{i-r+1}...c_i$)
:
\begin{itemize}
\item оставляем $c_i$ во входном потоке ($c_i$ "--- последний текущего слова и~первый следующего);
\item в~выходной поток пишем $P$.
\end{itemize}

\vfill

\item Декодирование: 
%номер шага $m$ = номер добавляемого узла.
%
перед любым шагом, кроме первого, последний узел (с~номером $m-1$) "--- пустой (с~неизвестным символом $c_j$);
%
читаем $P \in \{0, 1, ..., m-1\}$ $\implies$ пишем в~выходной поток слово = ветвь $P$ и~одновременно заполняем пустой узел:
\begin{itemize}
\item символ $c_{j}$, соответствующий первому (прикорневому $\implies$ всегда заполненному) узлу ветви $P$ пишем и~в~выходной поток, и~в~пустой узел $m-1$ (первый слова = последний предыдущего);
% $\implies$ 
% теперь все узлы заполнены;
% нет пустых;
\item {} [теперь нет пустых узлов] прочие символы ветви $P$, включая сам $P$ "--- только в выходной поток;
% "--- тепер
\item добавляем узлу $P$ пустой (с~неизвестным $c_i$) дочерний с~номером $m$.
% \mbox{(и~$m \to m+1$ для следующего шага).}
\end{itemize}

% \vfill

Первый шаг ($m=2^k$) отдельно: $P \in \{0, 1, ..., 2^k-1\}$, пустого нет.


\vfill

\item $|m|>>|c|$, во многих реализациях увеличивается по битам.

\item Дерево часто разворачивается в~таблицу.

\item Вх-й поток всегда дополняется как минимум одним незначащим \rlap{символом.}

\end{enumerate}


\end{adjustwidth}
\end{frame}
