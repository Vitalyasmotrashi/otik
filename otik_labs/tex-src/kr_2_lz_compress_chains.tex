Команде из 4 студентов необходимо реализовать по одному коду (задаются преподавателем) из всех трёх семейств: RLE (кроме наивного), LZ77, LZ78;
% \\
\textbf{из 3 студентов "--- два кода из разных семейств;}
 из 1-2 "--- один код.
% 
Для заданного сообщения $m$ и~каждого кода $code$ рассчитать:


\begin{enumerate}
\item длину сообщения $|m|$ в~символах;

\item для каждого кода $code$ "--- код сообщения $code(m)$ 
и~длину кода сообщения $|code(m)|$ в~символах (для алгоритмов с~битовым выходным потоком может быть нецелой;
для алгоритмов семейства LZ78 оценивать по минимуму: $|P|$ растёт по одному биту от нуля).

\end{enumerate}

\vfill

Для каждого кода: выбрать детали кодирования, не заданные преподавателем; указать выбранное и~следствия из него, в~частности:
\begin{enumerate}
\item %для алгоритмов семейства
 RLE:
формат записи сжатой цепочки $\{L, c\}$, $L_{\min}^{\text{сж}}$, $L_{\max}^{\text{сж}}$ +
\begin{itemize}
\item RLE с~флаг-битом сж/несж: + запись несжатой цепочки, $L_{\min}^{\text{несж}}$, $L_{\max}^{\text{несж}}$;
\item RLE с~односимвольным префиксом: + префикс $p$; запись $p$, $pp$ и~$ppp$;
\end{itemize}

% % \begin{enumerate}
% % \item длину 
% % \end{enumerate}
% \item RLE с~флаг-битом сжатая/несжатая цепочка: формат записи; 
% $L_{\min}^{\text{сж}}$, $L_{\min}^{\text{несж}}$,
% $L_{\max}^{\text{сж}}$, $L_{\max}^{\text{несж}}$;


\item  LZ77: формат ссылки $\{S, L\}$ (не только поля и~их порядок, но~и~разрядность, если символ состоит из [частей] разных полей "--- как), 
$S_{\min}$, $S_{\max}$, $L_{\min}$, $L_{\max}$ для ссылки;
\begin{itemize}
\item LZ77 с~флаг-битом ссылка/символ или ссылка/цепочка: + группируются ли флаг-биты во флаг-байты, запись несж. символа или цепочки;

\item LZ77 с~флаг-битом ссылка/цепочка: + запись несж. цепочки, \rlap{$L_{\min}^{\text{несж}}$, $L_{\max}^{\text{несж}}$;}
\item LZ77 с~односимвольным префиксом: + префикс $p$, запись символа $p$ \rlap{в~исх.\,тексте;}

\end{itemize}

\item длину кода наилучшего случая ($00...00$) длины  $|m|$, для RLE и~LZ77 + наихудшего длины  $|m|$.

\end{enumerate}
