
  
Дерево оптимального префиксного кода должно быть сбалансированным, что выражается одновременно в~трёх свойствах:

\begin{anumerate}
\item\label{item_tree_s} длина ветви листа $a$ в~рёбрах должна стремиться к~$I_{\text{БП}}(a)$ в~битах;

\item\label{item_tree_sf} для каждого родительского узла веса дочерних ветвей должны стремиться к~равенству между собой;

\item\label{item_tree_hf} самые длинные ветви (длинные коды) соответствуют самым редким листам (следует из \ref{item_tree_s}).
\end{anumerate}
Код Шеннона основан на \ref{item_tree_s}, Шеннона"--~Фано "--- на \ref{item_tree_sf}, Хаффмана "--- на \ref{item_tree_hf}.
